# Surveillance-Qualit-de-l-Air---IoT-LoRaWAN-
Ce projet consiste en la mise en place d'une chaîne complète d'acquisition et de transmission de données environnementales. L'objectif est de mesurer la qualité de l'air (CO2, Température, Humidité) et de transmettre ces informations via un réseau basse consommation pour une visualisation en temps réel.
